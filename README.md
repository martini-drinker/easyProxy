# EasyProxy
![Popup](./popup.png)

Simple firefox proxy switcher.
Firefox for Android support.

Support proxy: HTTP, HTTPS, SOCKS4, SOCKS5.
- Tab proxy
- Host list proxy (subdomain support)
- DNS proxy (SOCKS4, SOCKS5)
- Proxy authorization
- Host monitor
